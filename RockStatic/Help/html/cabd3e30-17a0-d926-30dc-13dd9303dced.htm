MyDicom Class
Wrapper para la libreria EvilDICOM (Rex Cardan, http://www.rexcardan.com/evildicom/) con modificaciones propias para ser manejadas por la suite RockUIS Un objeto MyDicom maneja un objeto de DICOM según la estructura de EvilDicom. Se incluyen los metodos para modificar el DICOM y guardarlo en disco
